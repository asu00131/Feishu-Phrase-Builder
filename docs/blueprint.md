# **App Name**: 数据看板

## Core Features:

- 动态数据展示: 从提供的 Feishu API 地址 ('https://bea9ijqf1k.feishu.cn/wiki/US1ewpweWiIHc0kLYR1cL8vQnOf?table=tblmipyCDtspHwa4&view=vewMND8m0m') 获取数据，并在分页表格中清晰展示所有列和行。
- 基本数据筛选: 允许用户根据特定列的值筛选表格数据行。
- 列排序功能: 通过点击列标题，支持表格数据的升序和降序排序。
- 中文界面支持: 应用程序的用户界面（包括标签、按钮和系统消息）提供全面的中文支持。
- 数据洞察工具: 利用生成式AI工具分析当前显示在表格中的数据，并提供简洁的摘要或发现有趣的模式。

## Style Guidelines:

- 主色: 一种平静而专业的蓝色 (#2E73B8)，象征着数据的清晰与信任。
- 背景色: 极浅的蓝灰色 (#ECF3F7)，提供清晰的底色，同时保持界面的柔和与专业感。
- 强调色: 鲜明的青色 (#4FDCD2)，用于交互元素和重要提示，确保视觉吸引力。
- 正文和标题字体: 'Inter'（无衬线字体），提供现代、中立且易于阅读的界面体验。
- 使用简洁、线条型的图标，符合现代数据管理应用的风格，增强表格筛选和排序等功能的可用性。
- 采用响应式布局，确保表格数据在不同屏幕尺寸（从桌面到移动设备）上都能良好地显示和操作。主界面以数据表格为中心，简洁且易于导航。
- 应用细微且流畅的动画效果，如表格排序、筛选时的过渡，以及分页切换时的柔和变化，提升用户体验。
